# ☕ Artisan Coffee Website

A premium coffee-themed website built with **Next.js + Framer Motion + Tailwind CSS**.  
Smooth animations, scroll storytelling, stylish sections, and a warm cafe vibe. ✨

---

## 🌟 Highlights

- 🎬 Scroll-based hero canvas animation (frame-by-frame coffee visuals)
- 🧾 Clean menu cards with product images and Indian pricing (`₹`)
- 🎯 Sticky top navigation with smooth scroll-to-section behavior
- 🖼️ Feature section with animated center canvas
- 🌰 Soft coffee-beans background blend in splash area
- 📱 Fully responsive modern UI

---

## 🛠️ Tech Stack

- **Next.js (App Router)**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

---

## 📂 Project Structure (Quick View)

- `app/` → page + global app layout
- `components/` → reusable UI sections (Hero, Menu, Features, CTA, Cards)
- `data/` → product + feature content data
- `public/` → frame sequences and static images

---

## 🚀 Getting Started

### 1) Install dependencies

```bash
npm install
```

### 2) Run development server

```bash
npm run dev
```

### 3) Open in browser

Visit: [http://localhost:3000](http://localhost:3000)

---

## 🧪 Build for Production

```bash
npm run build
npm start
```

---

## 🎨 Customization Tips

- Update menu items and prices in `data/products.ts`
- Update section copy in component files under `components/`
- Replace images in `public/` (keep same file names if you want zero code changes)
- Tweak color palette in `tailwind.config.ts` and `app/globals.css`

---

## ❤️ Notes

- This project is crafted for a **smooth visual coffee experience**.
- Keep image dimensions optimized for better performance.
- For frame animations, ensure naming consistency inside frame folders.
